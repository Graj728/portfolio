# Gaurav Raj Sah Portfolio Website

## Overview

This is a personal portfolio website built using HTML and CSS. The website showcases my background, technical skills, projects, education, and contact information. It serves as an online resume and a platform to present my work and achievements as a Computer Science student and aspiring Software Engineer.

## Features

* Responsive navigation bar
* Hero section with introduction
* About Me section
* Skills showcase
* Project portfolio
* Education and experience timeline
* Contact information
* Social media links (GitHub and LinkedIn)
* Clean and modern user interface
